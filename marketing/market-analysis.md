# はらぺこループ 市場分析レポート

**調査日**: 2026-03-21
**分析目的**: 同じようなアプリの有無確認、市場ポジションの把握

---

## 1. エグゼクティブサマリー

- 「空腹→満腹のループを共有するSNS」に一致する競合は**国内・海外ともにゼロ**
- 「空腹の共感・共有」は検索・SNSともに**完全な空白地帯**で、先行者利益を取れる
- 「平和なSNS」需要は実証済み（GRAVITY: 600万DL）。食事特化はまだ誰もやっていない

## 3. 競合分析

### 直接競合（食事 × SNS）

| サービス | ユーザー数 | 特徴 | はらぺこループとの違い |
|---------|-----------|------|---------------------|
| **SnapDish** | 月間406万人 | 料理写真SNS、AIカメラ | 映え写真の場。空腹の共感なし |
| **ミイル** | 縮小傾向 | 料理写真日記 | 食事記録メイン。交流薄い |
| **Pepper**（海外） | 約100万人 | ソーシャルクックブック | レシピ管理が目的 |

### 間接競合（ダイエット系）

| サービス | ユーザー数 | 特徴 | はらぺこループとの違い |
|---------|-----------|------|---------------------|
| **あすけん** | 累計1,300万人 | AI栄養管理 | 管理ツール。楽しさ・共感ではない |
| **MyFitnessPal** | MAU 3,000万人 | カロリー管理 | ガチ層向け。ライトユーザーには重い |
| **ミュベル** | 小規模 | 女性限定ダイエットSNS | ダイエット目的限定 |

### 間接競合（平和なSNS）

| サービス | ユーザー数 | 特徴 | はらぺこループとの違い |
|---------|-----------|------|---------------------|
| **GRAVITY** | DL 600万 | 匿名やさしいSNS | 汎用テーマ。食特化ではない |
| **DYSTOPIA** | 初期5万→現在50人 | AI検閲で平和を担保 | 話題性だけでは持続しない教訓 |
| **mixi2** | 約143万人 | 招待制クローズドSNS | 汎用SNS |

### ポジショニングマップ

```
        SNS性（高）
            |
    mixi2   |   SnapDish
    GRAVITY  |   ミイル
            |   Pepper
            |
機能シンプル ----+---- 機能リッチ
            |
    BeReal   |   あすけん
  ★はらぺこ  |   MyFitnessPal
   ループ    |   カロミル
            |
        SNS性（低）
```

**「機能シンプル × SNS性あり」の象限に直接の競合がいない。**

## 4. Porter's Five Forces

| 力 | 評価 | 理由 |
|----|------|------|
| **新規参入の脅威** | 中 | 技術的ハードルは低い。ただしコンセプトの模倣は難しい（構造的平和の設計） |
| **代替品の脅威** | 高 | X・Instagramで「#飯テロ」「#ダイエット垢」が既に機能している |
| **買い手の交渉力** | 高 | 無料SNSが大量にあり、ユーザーのスイッチングコストは低い |
| **売り手の交渉力** | 低 | インフラ（Supabase, Cloudflare）は無料〜低コスト |
| **業界内の競争** | 低 | 「空腹の共感SNS」カテゴリには競合がいない |

## 5. SWOT分析

| | ポジティブ | ネガティブ |
|---|-----------|-----------|
| **内部** | **Strengths** | **Weaknesses** |
| | 競合ゼロのユニークコンセプト | 開発者が非エンジニア（開発速度の制約） |
| | 構造的に荒れない設計（コメントなし） | マネタイズポイントが限られる |
| | 投稿ハードルが極めて低い（テキストのみ可） | ブランド認知がゼロ |
| | 運用コストほぼゼロ（全サービス無料枠） | 投稿の単調さが飽きにつながるリスク |
| **外部** | **Opportunities** | **Threats** |
| | 「平和なSNS」需要は実証済み（GRAVITY等） | 大手SNSが類似機能を追加する可能性 |
| | #飯テロ・#ダイエット垢の巨大な既存コミュニティ | DYSTOPIA教訓: 話題性だけでは持続しない |
| | 空腹時のデリバリーアフィは導線が自然 | 先行事例がない ＝ 需要がないリスク |
| | 「お腹減った 共感」SEOは完全空白 | 「はらぺこ」が大食いYouTuberと混同される可能性 |

## 6. 検索環境

### キーワード別の状況（2026-03-21調査）

| キーワード | 上位の競合 | はらぺこループの機会 |
|-----------|-----------|-------------------|
| 食事 SNS アプリ | SnapDish、アプリブ記事 | ランキング記事への掲載を狙う |
| お腹減った 共感 | 健康記事のみ | **完全な空白地帯** |
| ダイエット SNS | ミュベル、みんなのダイエット | 「空腹を楽しむ」で差別化 |

## 8. 戦略提言

### 今やるべき
1. **「はらぺこループ」で検索1位を取る** — note記事・LP公開でブランドSEO確立
2. **#お腹減った を自分たちのタグにする** — X投稿で旗を立てる
3. **DYSTOPIAの二の舞を避ける** — 話題性だけでなく、継続利用の仕組み（ループの完成感、リアクションの嬉しさ）を磨く

### 次にやるべき
4. **既存コミュニティへの乗り入れ** — #飯テロ #ダイエット垢 ユーザーへの訴求
5. **note記事で開発ストーリーを書く** — 「非エンジニアがSNS作った話」は検索にも強い
6. **デリバリーアフィの導線設計** — 空腹投稿時に「出前館で注文する？」的なUI

### 守るべき
7. **構造的平和を崩さない** — コメント機能追加の誘惑に負けない
8. **シンプルさを維持する** — 機能追加は慎重に。2種類の投稿がコア価値
