# X投稿ガイド（はらぺこループ）

## ツイートフレーミング
- **1行目で好奇心を作る**: 「もっと見る」をクリックさせる。結論や一番インパクトのある事実を冒頭に持ってくる
- 「Day X。」の後の1行目が勝負。ここで手を止めさせる
- 悪い例: 「今日はPWA対応をしました」→ 良い例: 「自分のスマホのホーム画面に、自分が作ったアプリのアイコンが並んでる。」

## 投稿後のアクション
- **コメントには必ず返信する**: コメント返信 = いいねの最低75倍のブースト効果
- 返信しないとアルゴリズムが「アクティブでない」と判断し、リーチが減る
- 丁寧な長文でなくてOK。短くても返す

## 引用RT（リポスト）の活用
- 過去のDay投稿を引用RTして「振り返り」投稿にする
- 引用のインプレッションは元投稿のインプレッション合計にカウントされる
- タイミング: Day7の振り返り時にDay1〜3を引用RTすると効果的

## インスピレーションバンク
- バイラルした個人開発系ツイートをXのブックマークフォルダ「個人開発バイラル」に保存する
- 投稿作成時に3〜5の類似バイラルツイートを参照する
- チェックポイント: 大文字か小文字か？エピソードか単刀直入か？何が目を止めさせたか？
